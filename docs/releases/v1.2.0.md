## Highlights

- **내보내기 프리셋 시스템 도입**: 슬롯 4개에 워터마크·합성·포맷 설정을 통째로 저장하고 한 번에 전환. 무료 2 슬롯 / Pro 4 슬롯, 활성 슬롯이 곧 글로벌 설정으로 자동 동기화.
- **페어 미리보기·메뉴 흐름 전면 개편**: 미리보기 모달에 캡슐 글래스 액션 바·핀치 줌 focal-point 와 드래그 패닝·광고 위치 재배치. 페어카드 롱탭 메뉴는 "재촬영" 을 "After 삭제" 로 교체해 합성본·기록 보존 채로 After 만 비울 수 있음.
- **인앱 리뷰 prompt**: 3회차 실행에서 자동 노출 + 설정 메뉴 "리뷰 남기기" 항목. 썸네일 표시 즉시성도 opportunistic delivery 로 개선.

## Added

- 내보내기 프리셋 4 슬롯 — 1번 "기본" 자동 시드 (삭제 불가), 슬롯 탭 시 활성 전환, 신규 생성 직후 자동 활성화, 중간 삭제 시 좌측 compaction + activeIndex 자동 보정, 이름 12자 제한 (`setting-preset`)
- 무료 사용자 프리셋 2 슬롯 접근 — 슬롯 2/3 은 Pro 잠금, 잠금 슬롯 탭 시 paywall 트리거 (`setting-preset · paywall`)
- 페어 미리보기 하단 캡슐 액션 바 — 공유 / 기기에 저장 / After 삭제 / 페어 삭제 4 동작 가로 배치, 광고 배너 상단 이동 (`pair-preview`)
- 페어 미리보기 핀치 줌 focal-point + 드래그 패닝 — 탭한 지점을 기준으로 확대, 줌인 후 다른 영역 탐색 가능, 더블 탭 reset (`pair-preview`)
- 페어카드 롱탭 메뉴 "After 삭제" — 합성본 자산·ExportHistory·hasCombinedExport 는 보존하고 After 원본만 정리. PhotoKit 시스템 삭제 권한 alert 만 사용 (`pair-card`)
- 인앱 리뷰 prompt — 3회차 실행 + 미요청 + 튜토리얼 비활성 조건에서 홈 첫 표시 onAppear 자동 호출, 한 번 호출 후 didRequestReview 영속 (`app-review`)
- 설정 → 도움말 → "리뷰 남기기" 진입점 — App Store write-review deep link (`app-review`)
- 튜토리얼 step 5 (프리셋 카드) + step 6 (1번 슬롯) — PreferenceKey 분리로 중첩 충돌 회피 (`setting-preset`)

## Changed

- 페어 미리보기 모달 시트 — 단일 detent `.fraction(0.7)` 로 사이즈 조절 비활성화, X 닫기 버튼 제거 (쓸어내려 닫기), 광고 배너 하단 → 상단 (`pair-preview`)
- 설정 화면 "저장공간 및 앱 정보" 섹션 → "정보" 한 단계 깊이로 정리 — 라이선스 / 사업자 정보 / 처리방침 / 이용약관 / 광고추적옵션 모두 새 InfoView 로 이동 (`settings`)
- 워터마크 logo 저장 방식 — JSON base64 직렬화 → Application Support 파일 분리 (`Watermark/<uuid>.dat`). 매 persist 마다 수 MB 인코딩 비용 제거. 기존 `logoImageData` 자동 마이그레이션, 저장 실패 시 다음 launch 자동 재시도 (`watermark`)
- 썸네일 미리보기 옵션 — `deliveryMode` opportunistic + `resizeMode` fast 로 완화. OS 캐시된 thumbnail 이 있으면 재방문 시 대기 없이 즉시 표시 (`thumbnail`)
- paywall 베네핏 한 줄 — "합성·워터마크 모든 기능 액세스" → "합성·워터마크·내보내기 전체 기능 액세스" (한·영) (`paywall`)
- StoreKit Configuration 가격 — 출시 가격에 맞춤 (Monthly ₩3,300 · 2주 무료 / Annual ₩33,000 무료체험 없음 · 17% 할인 표기), displayName/description 도 App Store 메타데이터와 일치 (`paywall`)
- 사업자 상호명 — 한·영 모두 "nomadLabs" 로 통일 (`settings`)
- Pro 만료 시 자동 강등 정합 — `activeIndex ≥ freeAccessibleSlotCount` 면 0 fallback, `format == .zip` 이면 individualImages 로 강등. SubscriptionStore.refresh / TransactionListener 콜백에서 자동 호출 (`setting-preset · subscription`)

## Fixed

- 페어 미리보기 핀치 줌이 항상 사진 중앙 기준으로만 확대되고 줌인 후 다른 부분을 못 보던 문제 — focal-point 수식 + 드래그 패닝 + (scale-1) × frameSize/2 경계 클램프 (`pair-preview`)
- `ExportJobBuilder.renderIndividual` 에서 워터마크 logoData 가 누락되어 개별 이미지 워터마크가 빠지던 회귀 (`export · watermark`)

## Removed

- 페어 미리보기 자체 "After 사진을 삭제할까요?" 확인 alert — PhotoKit 시스템 삭제 권한 alert 만 사용 (`pair-preview`)
- WatermarkSettings.`logoImageData(Data)` 필드 — `logoImageRef(String)` 로 대체 (자동 마이그레이션) (`watermark`)
