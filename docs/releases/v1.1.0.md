## Highlights

- **합성 설정 — 레이블 배치 방식 신설**: Before/After 라벨을 이미지 안쪽 (기존) 또는 테두리 안쪽에 배치할 수 있도록 옵션 추가. 테두리 모드에서는 6 위치 (좌·중·우 × 상·하) picker 로 라벨 위치 직접 지정.
- **인터랙션 강화**: AFTER 활성 카드 long-press 미리보기 + 튜토리얼 hint step, 스낵바 v5 재설계 (의미 글리프·title/body 카피), 홈 헤더 Pro 배지 레이아웃 정돈.
- **출시 안정화**: ATT → 권한 → UMP 순서 정공화 + UMP 와 ATT 분리 노출 (Pro 무관 무조건), 사업자 정보 표시 (한국 전자상거래법 의무), Pro 사용자 paywall 진입 — App Store 심사 2.1 재거절 차단.

## Added

- 합성 설정에 레이블 배치 방식 (이미지 내부 / 테두리 내부) 옵션 추가 — Before/After 각각 6위치 picker, 테두리 off 시 dim + snackbar 안내 (`combine-settings`)
- 사진 페어 카드의 AFTER long-press 미리보기 + 튜토리얼 hint step (`after-peek`)
- 사업자 정보 표시 섹션 — 한국 전자상거래법 의무 (`settings`)
- Pro 활성 사용자도 paywall 진입 가능한 항목 — 심사 거절 2.1 대응 (`settings`)
- 라벨 배경 불투명도 → 투명도 라벨 전환 (`combine`)

## Changed

- 스낵바 v5 재설계 — enum reason + title/body 카피 + 의미 글리프 UI (`snackbar`)
