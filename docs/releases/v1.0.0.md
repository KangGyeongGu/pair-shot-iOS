## Highlights

PairShot iOS 첫 출시. 현장 작업의 Before·After 페어 사진을 회전 가이드와 반투명 오버레이로 동일한 구도로 촬영하고, 앨범 관리·합성·다양한 내보내기까지 한 앱에서 처리할 수 있습니다. Pro 구독으로 페어 카드 무제한 + 광고 제거를 제공합니다.

---

## 주요 기능

- **Before/After 페어 촬영** — 반투명 오버레이로 동일한 구도의 작업 전후 사진 촬영. 핀치 줌, 렌즈 전환, 플래시, EV, 그리드, 수평계 지원
- **실시간 회전 가이드** — 핸드폰 아이콘 회전 애니메이션으로 기기 회전 방향 안내
- **페어 카드 / 앨범 관리** — SwiftData 기반, 작업 단위로 페어를 다중 앨범에 묶어 정리
- **비교·합성 화면** — 단일 모드 비교 + 합성 출력
- **내보내기** — 개별 이미지 / ZIP 묶음 / Before·After 합성 / 워터마크 / 진행률 표시
- **외부 갤러리 sync** — 시스템 사진 앱에서 원본 삭제 시 페어카드 자동 반영
- **광고** — Banner / Interstitial / AppOpen / Rewarded / Native (UMP consent + ATT 통합)
- **Pro 구독** — Monthly ₩5,500 / Annual ₩55,000 (약 17% 할인). 페어 카드 무제한 + 광고 제거
- **튜토리얼** — 메인 16-step + 내보내기 4-step. skip 확인 + 도움말 재실행
- **설정** — 다국어 (한국어 / 영어), 라이트·다크 테마, 이미지 품질 (기본 무손실 HEIC), 파일명 프리픽스, 워터마크, 합성 옵션
- **권한 처리** — 카메라 / 사진 / 위치 / ATT 순차 안내
